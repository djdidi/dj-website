## 定义

HTML（HyperText Markup Language，超文本标记语言）是一种描述语言，用来定义网页结构。

HTML 由一系列的元素组成，这些元素可以用来包围或标记不同部分的内容，使其以某种方式呈现或者工作。

- 超文本：是指连接单个网站内或多个网站间的网页的链接。
- 标记：HTML 使用“标记”（markup）来注明文本、图片和其他内容，以便于在 Web 浏览器中显示。
- 语言：通过书写标签（tag）来实现功能。
