# 正则表达式 RegExp

## 构造正则对象

- 字面量 `/.+/`
- 构造函数 `new RegExp(/ab+c/)` 一般动态内容时使用

具体语法主要为匹配字符、捕获括号、修饰符。

## 常用匹配字符

**转义**

在正则中有意义的字符需要 `\` 转义

**模式与修饰符**

- `/模式/修饰符`
- `new RegExp(模式, 修饰符);`

**字符类**

- `\d` `\w` `\s` 匹配某类字符，大写时表示不匹配
- `.` 匹配非换行字符，有修饰符 `s` 时，匹配任意字符

**锚点相关**

与修饰符 `m` 多行匹配影响

- `^` 输入开始
- `$` 输入结束

**集合、范围**

- `[abc]`
- `[a-zA-Z0-9]` 范围
- `[^abc]` 反向
- `[a|b]` or

**匹配次数**

前面字符出现次数

- `?` 0-1 次，等价于 `{0, 1}`
- `+` 1-n 次，等价于 `{1,}`
- `*` 0-n 次，等价于 `{0,}`
- `{n, m}` n-m 次

**捕获括号**

- `(x)` 记住匹配项。`\1...n` 匹配时复用，`$1...n` 替换时复用。
- `?<name>` 命名。示例：`/(?<year>[0-9]{4})/`

## 常用方法

RegExp 和 String 都有函数可以执行正则搜索操作。可以根据实际需求从简单到复杂分成三层。

1. 判断有无匹配；
2. 获取首个匹配；
3. 获取全部匹配。

**RegExp**

- test：判断有无匹配
- exec：返回匹配的详细搜索结果

**String**

- match / matchAll
- search
- replace / replaceAll
- split

**小结**

- test、search 用来判断是否匹配；
- exec、match、matchAll 用来获取匹配结果；
- replace、replaceAll 用来做替换。
