## IoT 基础概念

### 什么是物联网

IoT（即物联网）一词是指互联设备的集合网络，以及促进设备与云之间以及设备自身之间通信的技术。

### 工作原理

一般的 IoT 系统通过实时收集和交换数据来运作。主要有三个组件：

1. 智能设备
智能设备从环境、用户输入或使用模式中收集数据，通过网络与 IoT 应用程序进行数据通信。

2. IoT 应用程序
IoT 应用程序是一种服务和软件集合，整合了从各种 IoT 设备接收的数据。 与 IoT 设备进行链接通信，设备对输入进行响应。

3. 图形用户界面
用户可以通过图形用户界面管理 IoT 设备或设备机群。常见示例包括用于注册和控制智能设备的移动应用程序或网站。

### 配网

物联设备链接网络，主要分为两部分：
- 连接：Wi-Fi设备获取路由器信息（SSID、password等）并连接路由器的过程
- 绑定：应用端相关绑定

### MQTT

MQTT（Message Queuing Telemetry Transport）是一种轻量级、基于发布-订阅模式的消息传输协议，适用于资源受限的设备和低带宽、高延迟或不稳定的网络环境。

它在物联网应用中广受欢迎，能够实现传感器、执行器和其它设备之间的高效通信。

主要特点：
- 轻量级
- 可靠
- 安全通信
- 双向通信
- ...

### 固件升级 OTA

空中下载技术（Over-the-Air Technology， OTA）即通过移动通信的接口从远程服务器下载新的软件更新包，对自身系统进行升级，从而满足厂商的应用管理需求。
