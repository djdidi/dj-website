## 定义

**HTML**（超文本标记语言——HyperText Markup Language）是构成 Web 世界的一砖一瓦。它定义了网页内容的含义和结构。

- 超文本：是指连接单个网站内或多个网站间的网页的链接。
- 标记：HTML 使用“标记”（markup）来注明文本、图片和其他内容，以便于在 Web 浏览器中显示。
- 语言：通过书写标签（tag）来实现功能。
